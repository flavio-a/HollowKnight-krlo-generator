What is a "random order generator"? You may have heard of Salubra's Blessing Random Charm Order (SBRCO, or "Bridie's obsession"). That is a prime example of random order generator. You start from any collectathon category (for instance Salubra's Blessing, where you collect all the charms) but instead of just collectting all the items as fast as possible, you are also given a randomly generated order in which you have to collect them. A random order generator is (you guessed it) a program that generates such a random order.
What is this website? This website is a collection of random order generators I (Ff300) have developed. They are all based on the same program, just using different logics and settings for the specific cases.
